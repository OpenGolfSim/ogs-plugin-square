A launch monitor plugin for OpenGolfSim that connects to the Square and reads shot data via bluetooth.

## Installation

To install this plugin, download this repo as a [ZIP file](https://github.com/OpenGolfSim/ogs-plugin-square/archive/refs/heads/main.zip) and extract the folder to your plugins folder location for your platform.

#### Windows

On Windows, the plugins folder should be found at:

```
%USERPROFILE%\AppData\Roaming\opengolfsim-desktop\plugins\
```

#### MacOS

On MacOS, the plugins folder should be found at:

```
~/Library/Application Support/opengolfsim-desktop/plugins/
```
